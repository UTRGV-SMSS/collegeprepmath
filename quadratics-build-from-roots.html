    Section 10.5 Build Quadratics From Roots

    Objective: Find a quadratic equation that has given roots using reverse factoring and reverse completing the square.
    Up to this point we have found the solutions to quadratics by a method such as factoring or completing the square. Here we will take our solutions and work backwards to find what quadratic goes with the solutions.
    We will start with rational solutions. If we have rational solutions we can use factoring in reverse, we will set each solution equal to x and then make the equation equal to zero by adding or subtracting. Once we have done this our expressions will become the factors of the quadratic.

    Example 10.5.1.

    \begin{align*} \text{The solutions are $4$ and } -2 \amp \quad \text{Set each solution equal to $x$} \\ x =4 \text{ or } x = - 2 \amp\quad \text{Make each equation equal zero } \\ \underline{- 4 \ – 4} \quad \underline{+2 \ \ + 2} \amp\quad \text{Subtract $4$ from first, add $2$ to second } \\ x – 4 = 0 \text{ or } x + 2= 0 \amp\quad \text{These expressions are the factors} \\ (x-4)(x+2)=0 \amp\quad \text{FOIL } \\ x^2+2x-4x-8=0 \amp\quad \text{Combine like terms } \\ x^2-2x-8=0 \amp\quad \text{ Our Solution } \end{align*}
    If one or both of the solutions are fractions we will clear the fractions by multiplying by the denominators.

    Example 10.5.2.

    \begin{align*} \text{The solutions are } \frac{2}{3} \text{ and } \frac{3}{4}\amp \quad \text{Set each solution equal to }x \\ x=\frac{2}{3} \text{ or } x=\frac{3}{4} \amp\quad \text{ Clear fractions by multiplying by denominators } \\ 3x=2 \text{ or } 4x=3 \amp\quad \text{ Make each equation equal zero } \\ \underline{-2-2} \quad \underline{ -3-3} \amp\quad \text{ Subtract $2$ from the first, subtract $3$ from the second } \\ 3x-2=0 \text{ or } 4x-3=0 \amp\quad \text{ These expressions are the factors } \\ (3x-2)(4x-3)=0 \amp\quad \text{ FOIL } \\ 12x^2-9x-8x+6=0 \amp\quad \text{Combine like terms } \\ 12x^2-17x+6=0 \amp\quad \text{Our Solution } \end{align*}
    If the solutions have radicals (or complex numbers) then we cannot use reverse factoring. In these cases we will use reverse completing the square. When there are radicals the solutions will always come in pairs, one with a plus, one with a minus, that can be combined into “one” solution using \(\pm\text{.}\) We will then set this solution equal to \(x\) and square both sides. This will clear the radical from our problem.

    Example 10.5.3.

    \begin{align*} \text{The solutions are } \sqrt{3} \text{ and } -\sqrt{3} \amp \quad \text{Write as “one” expression equal to } x \\ x=\pm \sqrt{3} \amp\quad \text{Square both sides } \\ x^2=3 \amp\quad \text{Make equal to zero} \\ \underline{-3-3} \amp\quad \text{Subtract $3$ from both sides } \\ x^2-3=0 \amp\quad \text{Our Solution } \end{align*}
    We may have to isolate the term with the square root (with plus or minus) by adding or subtracting. With these problems, remember to square a binomial we use the formula \((a + b)^2 = a^2 + 2ab + b^2\)

    Example 10.5.4.

    \begin{align*} \text{The solutions are } 2-5\sqrt{2} \text{ and } 2+5\sqrt{2} \amp \quad \text{Write as “one” expression equal to } x \\ x=2 \pm 5\sqrt{2}\amp\quad \text{Isolate the square root term } \\ \underline{-2\ -2 } \quad \qquad \amp\quad \text{Subtract $2$ from both sides} \\ x-2=\pm 5\sqrt{2} \amp\quad \text{Square both sides } \\ x^2-4x+4=25 \cdot 2 \amp\quad \text{ } \\ x^2-4x+4=50 \amp\quad \text{Make equal to zero } \\ \underline{-50-50} \amp\quad \text{Subtract 50 } \\ x^2-4x-46=0 \amp\quad \text{Our Solution } \end{align*}
    World View Note: Before the quadratic formula, before completing the square, before factoring, quadratics were solved geometrically by the Greeks as early as \(300\) BC! In \(1079\) Omar Khayyam, a Persian mathematician solved cubic equations geometrically!
    If the solution is a fraction we will clear it just as before by multiplying by the denominator.

    Example 10.5.5.

    \begin{align*} \text{The solutions are } \frac{2+\sqrt{3}}{4} \text{ and } \frac{2-\sqrt{3}}{4} \amp \quad \text{Write as “one” expression equal to} x \\ x=\frac{2\pm \sqrt{3}}{4}\amp\quad \text{Clear fraction by multiplying by } 4 \\ 4x=2\pm \sqrt{3} \amp\quad \text{Isolate the square root term} \\ \underline{-2\ -2\ }\qquad \amp\quad \text{Subtract $2$ from both sides } \\ 4x-2=\pm \sqrt{3} \amp\quad \text{Square both sides } \\ 16x^2-16x+4=3 \amp\quad \text{Make equal to zero} \\ \underline{-3-3} \amp\quad \text{ Subtract } 3 \\ 16x^2-16x+1=0 \amp\quad \text{Our Solution } \end{align*}
    The process used for complex solutions is identical to the process used for radicals.

    Example 10.5.6.

    \begin{align*} \text{The solutions are } 4+5i \text{ and } 4-5i \amp \quad \text{Write as “one” expression equal to} x \\ x=4\pm 5i \amp\quad \text{Isolate the $i$ term} \\ \underline{-4\ -4}\qquad \amp\quad \text{Subtract $4$ from both sides } \\ x-4=\pm 5i \amp\quad \text{Square both sides } \\ x^2-8x+16=25i^2 \amp\quad \text{}i^2=-1 \\ x^2-8x+16=-25 \amp\quad \text{Make equal to zero} \\ \underline{+25\quad +25}\amp\quad \text{ Add $25$ to both sides} \\ x^2 -8x +41 =0 \amp\quad \text{Our Solution } \end{align*}

    Example 10.5.7.

    \begin{align*} \text{The solutions are } \frac{3-5i}{2} \text{ and } \frac{3+5i}{2} \amp \quad \text{Write as “one” expression equal to} x \\ x=\frac{3\pm 5i}{2} \amp\quad \text{Clear fraction by multiplying by denominator } \\ 2x=3\pm 5i \amp\quad \text{Isolate the $i$ term} \\ \underline{-3\ -3}\qquad \amp\quad \text{Subtract $3$ from both sides } \\ 2x-3=\pm 5i \amp\quad \text{Square both sides } \\ 4x^2-12x+9=5i^2 \amp\quad \text{}i^2=-1 \\ 4x^2-12x+9=-25 \amp\quad \text{Make equal to zero} \\ \underline{+25\quad +25}\amp\quad \text{ Add $25$ to both sides} \\ 4x^2-12x+34=0 \amp\quad \text{Our Solution } \end{align*}
